
# Level Format

Level format taken from:

http://www.sokobano.de/wiki/index.php?title=Level_format

Summary:

| Element        | Char | 
| -------------- | ---- |
| Wall           | `#`  |
| Worker         | `@`  |
| Worker on goal | `+`  |
| Box            | `$`  |
| Box on goal    | `*`  |
| Goal           | `.`  |
| Floor          | ` `  |
